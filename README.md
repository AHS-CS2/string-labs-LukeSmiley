# ReadMe Demo

This is the demo file for this project. It is being used to demonstrate the new work flow for all assignments.

- Clone repo OR git pull --no-edit
- Create/Move into branch for specific lab.
- Stage changes and commit
- When lab is finsihed, push to remote --> git push origin [name of branch]
- Go to GitHub and open a new Pull Request
- Review autograder or feedback from Mr. Azam.
- When completed, Mr. Azam will merge the Pull Request
